# DMC Thread Reference Guide

## Thread Types and Sizes

### Mouliné Spécial (6-Strand Cotton Floss)
- Standard for cross stitch and needlepoint
- 6 separable strands
- Use 1-2 strands for fine work, 3-6 for coverage

### Pearl Cotton (Perle)
| Size | Diameter | Use |
|------|----------|-----|
| 3 | Thickest | Bold coverage, large canvas |
| 5 | Medium-thick | 10-14 count canvas |
| 8 | Medium | Detail work, 14-18 count |
| 12 | Finest | Fine detail, petit point |

### Tapestry Wool
- 4-ply divisible wool
- Best for 10-14 mesh canvas

### DMC Satin
Six-strand divisible floss with satin sheen. Available in 120 colors numbered S-prefix (S151, S209, etc.)

### DMC Diamant
Non-divisible metallic thread. Colors: D140, D168, D225, D301, D310, D316, D321, D380, D415, D699, D898, D3821, D3852

### DMC Coloris
Variegated 6-strand cotton with color transitions. Numbered 4000-4523.

## Canvas and Thread Pairing Guide

| Canvas Count | Recommended Thread | Strands |
|--------------|-------------------|---------|
| 7 mesh | Tapestry wool | Full strand |
| 10 mesh | Pearl Cotton #3 | N/A |
| 12 mesh | Pearl Cotton #5 | N/A |
| 13 mesh | 6-strand floss | 4-6 strands |
| 14 mesh | 6-strand floss | 3-4 strands |
| 18 mesh | 6-strand floss | 2-3 strands |
| 22+ mesh | 6-strand floss | 1-2 strands |

## Color Conversion Notes

DMC numbers serve as the industry standard. Most manufacturers provide conversion charts to their proprietary numbering systems.

Common conversions exist for:
- Anchor to DMC
- Weeks Dye Works to DMC
- Gentle Art to DMC
- Classic Colorworks to DMC
